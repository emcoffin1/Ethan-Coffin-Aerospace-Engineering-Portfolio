Project 04

Rocket Engine Generator and Regenerative Cooling Analysis

Thermal-fluid simulation / propulsion analysis | Developer / analyst

Overview

Python tool chain for early rocket engine sizing, nozzle performance, heat flux estimation, and regenerative cooling exploration.

Objective

Model engine performance and thermal behavior quickly enough to support concept trades and downstream vehicle simulation.

Process

  • Modeled compressible flow through Rao bell nozzles using isentropic relations and numerical methods.
  • Implemented regenerative-cooling models using convective heat transfer, thermal resistance networks, and Bartz heat-transfer estimates.
  • Predicted temperature, pressure, and heat-flux distributions along nozzle geometry.
  • Validated flow and heat-transfer predictions using Ansys Fluent CFD simulations.

Outcome / Evidence

  • Thermal analysis and propulsion skill pages list regenerative cooling, nozzle flow, and Fluent validation.
  • Shows ability to combine first principles, numerical methods, and CFD sanity checks.
